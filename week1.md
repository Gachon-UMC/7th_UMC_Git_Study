# Git-Study

## Week 1

### Git이란?

파일의 변경 사항을 추적하고 여러 사용자 간에 해당 파일들의 작업들을 조율하기 위한 분산 버전 관리 시스템

#### .gitignore 파일 생성

touch .gitignore

#### 특정 확장자를 가진 파일 추적 해제

\*.(확장자 이름)

#### 모든 파일 staged

git add .

#### 커밋

git commit -m "(커밋 메시지)"

#### 브랜치 생성

git branch (브랜치 이름)

#### 브랜치 이동

git switch (브랜치 이름)
